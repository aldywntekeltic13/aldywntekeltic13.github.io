# Starlight Derby Danmaku

A browser-playable Touhou/Danmaku-inspired bullet hell game starring Teio Tannhauser and Mambo Hachimi.

## Run

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
```

## Controls

- Arrow keys or WASD: move
- Shift: focus movement and show hitbox
- Z or Space: shoot
- X: bomb
- P or Esc: pause
- F1: debug overlay
- F2: hitbox overlay
- F3: slow motion
- F4: invulnerability

This is a free fan-game prototype using original/generated assets and no ripped official game assets.
